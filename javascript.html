Pilih Bagian

Kembali Pilih Materi

Pengertian

JavaScript adalah bahasa untuk membuat website interaktif.

Penjelasan

Digunakan untuk tombol, animasi, dan logika website.

Contoh

alert("Halo")

Kesimpulan

JavaScript membuat website jadi hidup.

Pengertian

Variabel digunakan untuk menyimpan data.

Penjelasan

Gunakan let atau const untuk membuat variabel.

Contoh

let nama = "Budi";

Kesimpulan

Variabel menyimpan data dalam program.

Pengertian

Tipe data adalah jenis nilai.

Penjelasan

Contohnya string, number, boolean.

Contoh

let umur = 15;

Kesimpulan

Setiap data punya jenisnya masing-masing.

Pengertian

Fungsi adalah blok kode yang bisa dipanggil.

Penjelasan

Digunakan agar kode lebih rapi dan bisa digunakan ulang.

Contoh

function halo(){
  alert("Halo");
}

Kesimpulan

Fungsi membuat program lebih terstruktur.

Pengertian

Event adalah aksi dari user.

Penjelasan

Contohnya klik tombol atau input data.

Contoh

button.onclick = function(){
  alert("Klik");
}

Kesimpulan

Event membuat interaksi di website.

Pengertian

Percabangan digunakan untuk mengambil keputusan.

Penjelasan

Menggunakan if, else untuk kondisi tertentu.

Contoh

if(umur > 17){
  alert("Dewasa");
}

Kesimpulan

Percabangan menentukan alur program.

Pengertian

Perulangan digunakan untuk mengulang kode.

Penjelasan

Menggunakan for atau while untuk looping.

Contoh

for(let i=0; i<3; i++){
  console.log(i);
}

Kesimpulan

Perulangan mempermudah pekerjaan berulang.

Pengertian

Array adalah kumpulan data dalam satu variabel.

Penjelasan

Digunakan untuk menyimpan banyak nilai.

Contoh

let buah = ["Apel", "Jeruk"];

Kesimpulan

Array menyimpan banyak data sekaligus.

Pengertian

Object adalah kumpulan data dengan properti.

Penjelasan

Berisi key dan value dalam satu variabel.

Contoh

let user = {
  nama: "Budi",
  umur: 15
};

Kesimpulan

Object menyimpan data lebih kompleks.

Pengertian

DOM adalah cara JS mengakses HTML.

Penjelasan

Digunakan untuk mengubah isi halaman.

Contoh

document.getElementById("judul").innerHTML = "Halo";

Kesimpulan

DOM membuat halaman bisa diubah.

Pengertian

Operator digunakan untuk melakukan operasi pada data.

Penjelasan

Contohnya operator aritmatika (+, -, *, /) dan perbandingan (>, <, ==).

Contoh

let hasil = 5 + 3;
let cek = 5 > 3;

Kesimpulan

Operator membantu melakukan perhitungan dan logika.

Pengertian

String adalah tipe data berupa teks.

Penjelasan

Ditulis dengan tanda kutip dan bisa digabungkan.

Contoh

let nama = "Budi";
let sapa = "Halo " + nama;

Kesimpulan

String digunakan untuk mengolah teks.

Pengertian

Number adalah tipe data angka.

Penjelasan

Bisa berupa bilangan bulat atau desimal.

Contoh

let umur = 15;
let tinggi = 170.5;

Kesimpulan

Number digunakan untuk perhitungan angka.

Pengertian

Boolean adalah nilai true atau false.

Penjelasan

Biasanya digunakan dalam percabangan.

Contoh

let login = true;

Kesimpulan

Boolean menentukan kondisi benar atau salah.

Pengertian

Selector digunakan untuk memilih elemen HTML.

Penjelasan

Bisa menggunakan id, class, atau tag.

Contoh

document.querySelector(".box");

Kesimpulan

Selector memudahkan akses elemen HTML.

Pengertian

JavaScript bisa mengubah tampilan CSS.

Penjelasan

Digunakan untuk membuat tampilan dinamis.

Contoh

document.body.style.background = "black";

Kesimpulan

JS bisa mengontrol tampilan halaman.

Pengertian

Input digunakan untuk mengambil data dari user.

Penjelasan

Bisa menggunakan prompt atau form HTML.

Contoh

let nama = prompt("Masukkan nama");

Kesimpulan

Input membuat program interaktif.

Pengertian

Output digunakan untuk menampilkan hasil.

Penjelasan

Bisa menggunakan alert, console, atau HTML.

Contoh

console.log("Halo Dunia");

Kesimpulan

Output menampilkan hasil ke user.

Pengertian

Array memiliki fungsi tambahan (method).

Penjelasan

Contohnya push, pop, dan length.

Contoh

let buah = ["Apel"];
buah.push("Jeruk");

Kesimpulan

Method mempermudah manipulasi array.

Pengertian

Event Listener mendeteksi aksi user.

Penjelasan

Lebih fleksibel dibanding onclick.

Contoh

document.addEventListener("click", function(){
  alert("Klik terdeteksi");
});

Kesimpulan

Event Listener membuat interaksi lebih lengkap.

Pengertian

Scope adalah jangkauan akses suatu variabel.

Penjelasan

Variabel bisa bersifat global (bisa diakses dimana saja) atau lokal (hanya dalam fungsi).

Contoh

let a = 10;
function test(){
  let b = 5;
}

Kesimpulan

Scope menentukan dimana variabel bisa digunakan.

Pengertian

Arrow function adalah cara singkat membuat fungsi.

Penjelasan

Ditulis menggunakan tanda => dan lebih ringkas dari function biasa.

Contoh

const halo = () => {
  console.log("Halo");
};

Kesimpulan

Arrow function membuat kode lebih sederhana.

Pengertian

Template string digunakan untuk menggabungkan teks dengan variabel.

Penjelasan

Menggunakan backtick (`) dan ${} untuk memasukkan variabel.

Contoh

let nama = "Budi";
let sapa = `Halo ${nama}`;

Kesimpulan

Template string mempermudah penulisan teks dinamis.

Pengertian

Destructuring adalah cara mengambil data dari array atau object.

Penjelasan

Membuat pengambilan data jadi lebih cepat dan rapi.

Contoh

let user = {nama: "Budi", umur: 15};
let {nama, umur} = user;

Kesimpulan

Destructuring mempermudah akses data.

Pengertian

Spread operator digunakan untuk menyalin atau menggabungkan data.

Penjelasan

Ditulis dengan ... dan sering dipakai pada array atau object.

Contoh

let a = [1,2];
let b = [...a, 3];

Kesimpulan

Spread operator membuat manipulasi data lebih mudah.

Pengertian

Callback adalah fungsi yang dikirim ke fungsi lain sebagai argumen.

Penjelasan

Callback dijalankan setelah fungsi utama selesai, biasanya untuk menunggu proses selesai.

Contoh

function halo(nama, cb){
  console.log("Halo " + nama);
  cb();
}
halo("Budi", function(){ console.log("Selesai"); });

Kesimpulan

Callback membuat kode bisa dijalankan berurutan.

Pengertian

Promise adalah objek untuk menangani operasi asynchronous.

Penjelasan

Promise bisa resolve (berhasil) atau reject (gagal) dan membantu menghindari callback hell.

Contoh

let janji = new Promise((resolve, reject) => {
  resolve("Sukses");
});
janji.then(res => console.log(res));

Kesimpulan

Promise memudahkan manajemen kode async.

Pengertian

Async/Await membuat kode async terlihat seperti synchronous.

Penjelasan

Async menandai fungsi async, Await menunggu hasil Promise.

Contoh

async function test(){
  let res = await fetch("data.json");
  console.log(res);
}
test();

Kesimpulan

Async/Await membuat kode lebih rapi dan mudah dibaca.

Pengertian

Local Storage menyimpan data di browser secara permanen.

Penjelasan

Data bisa diambil kapan saja selama belum dihapus, dan cocok untuk menyimpan setting user.

Contoh

localStorage.setItem("nama", "Budi");
let nama = localStorage.getItem("nama");

Kesimpulan

Local Storage mempermudah penyimpanan data kecil di browser.

Pengertian

Session Storage menyimpan data sementara di browser.

Penjelasan

Data hilang ketika tab atau browser ditutup, cocok untuk menyimpan data sementara.

Contoh

sessionStorage.setItem("login", "true");
let login = sessionStorage.getItem("login");

Kesimpulan

Session Storage cocok untuk data sementara dan bersifat session-based.

Pengertian

Fungsi untuk memilih elemen HTML berdasarkan id atau class.

Penjelasan

getElementById memilih satu elemen, getElementsByClassName memilih beberapa elemen sekaligus.

Contoh

let judul = document.getElementById("judul");
let boxes = document.getElementsByClassName("box");

Kesimpulan

Memudahkan akses elemen untuk dimanipulasi.

Pengertian

Selector modern untuk memilih elemen HTML.

Penjelasan

querySelector memilih elemen pertama, querySelectorAll memilih semua elemen sesuai selector CSS.

Contoh

let box = document.querySelector(".box");
let allBoxes = document.querySelectorAll(".box");

Kesimpulan

Lebih fleksibel dibanding getElementById/class.

Pengertian

JS bisa mengubah teks elemen HTML.

Penjelasan

Gunakan innerHTML atau textContent untuk menampilkan atau mengubah teks.

Contoh

document.getElementById("judul").textContent = "Halo Dunia";

Kesimpulan

Manipulasi teks membuat halaman lebih interaktif.

Pengertian

JS bisa menambah, menghapus, atau mengubah elemen HTML.

Penjelasan

Menggunakan innerHTML atau createElement & appendChild.

Contoh

let p = document.createElement("p");
p.textContent = "Paragraf baru";
document.body.appendChild(p);

Kesimpulan

Manipulasi HTML membuat halaman dinamis.

Pengertian

JS bisa membaca dan mengubah atribut elemen.

Penjelasan

Gunakan getAttribute, setAttribute, atau removeAttribute.

Contoh

let link = document.querySelector("a");
link.setAttribute("href", "https://example.com");

Kesimpulan

Mengubah atribut membuat elemen lebih fleksibel.

Pengertian

JS bisa menambah, hapus, atau toggle class CSS.

Penjelasan

Gunakan classList.add, remove, toggle untuk kontrol styling.

Contoh

let box = document.querySelector(".box");
box.classList.add("aktif");

Kesimpulan

Mengontrol class mempermudah styling dinamis.

Pengertian

Mendeteksi input keyboard dari user.

Penjelasan

Gunakan keydown, keyup, atau keypress untuk menangkap tombol yang ditekan.

Contoh

document.addEventListener("keydown", function(e){
  console.log("Tombol: " + e.key);
});

Kesimpulan

Event keyboard membuat interaksi lebih kompleks.

Pengertian

Mendeteksi aksi mouse seperti klik, hover, atau double click.

Penjelasan

Event mouse membantu membuat UI interaktif.

Contoh

let btn = document.querySelector("button");
btn.addEventListener("dblclick", function(){
  alert("Double klik!");
});

Kesimpulan

Event mouse mempermudah interaksi visual.

Pengertian

Mendeteksi aksi form seperti submit atau input.

Penjelasan

Membantu validasi data sebelum dikirim ke server.

Contoh

let form = document.querySelector("form");
form.addEventListener("submit", function(e){
  e.preventDefault();
  alert("Form disubmit");
});

Kesimpulan

Event form membuat data user lebih aman dan terkontrol.

Pengertian

Mendengarkan event pada parent untuk menangkap event anak.

Penjelasan

Memudahkan manajemen event untuk banyak elemen dinamis.

Contoh

document.body.addEventListener("click", function(e){
  if(e.target.classList.contains("box")){
    alert("Box diklik");
  }
});

Kesimpulan

Delegasi event membuat kode lebih efisien untuk elemen dinamis.

Pengertian

AJAX digunakan untuk mengambil data tanpa reload halaman.

Penjelasan

AJAX pakai XMLHttpRequest untuk request ke server dan menampilkan data secara dinamis.

Contoh

let xhr = new XMLHttpRequest();
xhr.open("GET", "data.json", true);
xhr.onload = function(){
  console.log(xhr.responseText);
};
xhr.send();

Kesimpulan

AJAX membuat website lebih interaktif tanpa reload.

Pengertian

Fetch API modern untuk request data dari server.

Penjelasan

Lebih sederhana dibanding XMLHttpRequest dan mendukung Promise.

Contoh

fetch("data.json")
  .then(res => res.json())
  .then(data => console.log(data));

Kesimpulan

Fetch membuat request data lebih mudah dan rapi.

Pengertian

JSON adalah format data yang mudah dibaca manusia dan komputer.

Penjelasan

Digunakan untuk pertukaran data antara client dan server.

Contoh

let user = {"nama":"Budi","umur":15};
console.log(user.nama);

Kesimpulan

JSON memudahkan pertukaran data di web.

Pengertian

JS bisa mengubah JSON menjadi object dan sebaliknya.

Penjelasan

Menggunakan JSON.parse untuk ubah JSON jadi object, JSON.stringify untuk ubah object jadi JSON.

Contoh

let data = '{"nama":"Budi"}';
let obj = JSON.parse(data);
console.log(obj.nama);

let json = JSON.stringify(obj);

Kesimpulan

Manipulasi JSON penting untuk data server-client.

Pengertian

LocalStorage bisa menyimpan object dan array dengan JSON.

Penjelasan

Menyimpan data besar dan mengambil kembali tanpa kehilangan format.

Contoh

let user = {nama:"Budi", umur:15};
localStorage.setItem("user", JSON.stringify(user));
let getUser = JSON.parse(localStorage.getItem("user"));

Kesimpulan

LocalStorage bisa simpan data kompleks dengan JSON.

Pengertian

SessionStorage bisa menyimpan data sementara dengan JSON.

Penjelasan

Data hilang saat tab ditutup, cocok untuk session user.

Contoh

let session = {login:true};
sessionStorage.setItem("sess", JSON.stringify(session));
let getSess = JSON.parse(sessionStorage.getItem("sess"));

Kesimpulan

SessionStorage cocok untuk data sementara berbasis sesi.

Pengertian

Mendeteksi perubahan input user secara real-time.

Penjelasan

Gunakan event input untuk menangkap perubahan value setiap saat user mengetik.

Contoh

let input = document.querySelector("input");
input.addEventListener("input", function(e){
  console.log(e.target.value);
});

Kesimpulan

Event input membuat interaksi lebih responsif.

Pengertian

JS bisa menambah, hapus, dan ubah baris di tabel HTML.

Penjelasan

Menggunakan createElement, appendChild, atau innerHTML untuk manipulasi tabel.

Contoh

let table = document.querySelector("table tbody");
let row = document.createElement("tr");
row.innerHTML = "1Budi";
table.appendChild(row);

Kesimpulan

Manipulasi tabel membuat data di web lebih dinamis.

Pengertian

JS bisa memvalidasi data sebelum dikirim ke server.

Penjelasan

Mengecek input kosong, format email, atau batas karakter.

Contoh

let form = document.querySelector("form");
form.addEventListener("submit", function(e){
  let email = form.email.value;
  if(email === "") e.preventDefault();
});

Kesimpulan

Validasi form membuat data user lebih aman dan rapi.

Pengertian

Timer digunakan untuk menjalankan kode setelah waktu tertentu.

Penjelasan

Gunakan setTimeout untuk sekali, setInterval untuk berulang.

Contoh

setTimeout(() => console.log("Delay 2 detik"), 2000);
setInterval(() => console.log("Setiap 3 detik"), 3000);

Kesimpulan

Timer membuat kode berjalan otomatis sesuai waktu.
