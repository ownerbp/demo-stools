Kembali Pilih Materi

Pengertian

Python adalah bahasa pemrograman yang mudah dipelajari.

Penjelasan

Python sering digunakan oleh pemula karena sintaksnya sederhana dan mudah dibaca.

Contoh

print("Halo Dunia")

Kesimpulan

Python cocok untuk belajar coding dari awal.

Pengertian

Variabel adalah tempat untuk menyimpan data.

Penjelasan

Di Python, kamu tidak perlu menentukan tipe data secara manual.

Contoh

nama = "Natan"

Kesimpulan

Variabel digunakan untuk menyimpan nilai dalam program.

Pengertian

Tipe data adalah jenis dari data yang digunakan.

Penjelasan

Contohnya seperti string, integer, float, dan boolean.

Contoh

nama = "Bintang"
umur = 15

Kesimpulan

Python memiliki berbagai tipe data untuk kebutuhan berbeda.

Pengertian

Input dan output digunakan untuk interaksi dengan user.

Penjelasan

Input untuk menerima data, output untuk menampilkan data.

Contoh

nama = input("Nama: Brian")
print(nama)

Kesimpulan

Program bisa menerima dan menampilkan data menggunakan I/O.

Pengertian

If digunakan untuk membuat keputusan.

Penjelasan

Program akan menjalankan kode jika kondisi terpenuhi.

Contoh

if umur > 17:
  print("Dewasa")

Kesimpulan

If digunakan untuk logika kondisi dalam program.

Pengertian

Else dan elif digunakan untuk kondisi tambahan dalam percabangan.

Penjelasan

Elif digunakan untuk menambah kondisi lain, sedangkan else dijalankan jika semua kondisi salah.

Contoh

umur = 16

if umur > 17:
  print("Dewasa")
elif umur == 17:
  print("Menuju Dewasa")
else:
  print("Anak-anak")

Kesimpulan

Else dan elif membantu membuat keputusan lebih kompleks.

Pengertian

Perulangan for digunakan untuk mengulang kode dengan jumlah tertentu.

Penjelasan

Biasanya digunakan untuk mengulang data seperti list atau range.

Contoh

for i in range(5):
  print("Perulangan ke-", i)

Kesimpulan

For digunakan untuk perulangan yang sudah diketahui jumlahnya.

Pengertian

While digunakan untuk perulangan berdasarkan kondisi.

Penjelasan

Perulangan akan terus berjalan selama kondisi bernilai benar.

Contoh

i = 0
while i < 5:
  print(i)
  i += 1

Kesimpulan

While cocok untuk kondisi yang belum pasti jumlah perulangannya.

Pengertian

List adalah kumpulan data dalam satu variabel.

Penjelasan

List bisa menyimpan banyak nilai dan bisa diakses menggunakan index.

Contoh

buah = ["apel", "jeruk", "mangga"]
print(buah[0])

Kesimpulan

List digunakan untuk menyimpan banyak data sekaligus.

Pengertian

Function adalah blok kode yang bisa digunakan kembali.

Penjelasan

Function membantu membuat kode lebih rapi dan efisien.

Contoh

def sapa():
  print("Halo Rapi")

sapa()

Kesimpulan

Function membuat program lebih terstruktur.

Pengertian

Parameter adalah input yang diberikan ke dalam function.

Penjelasan

Dengan parameter, function bisa menerima nilai berbeda.

Contoh

def sapa(nama):
  print("Halo", nama)

sapa("Akbar")

Kesimpulan

Parameter membuat function lebih fleksibel.

Pengertian

Return digunakan untuk mengembalikan nilai dari function.

Penjelasan

Nilai yang dikembalikan bisa disimpan ke variabel lain.

Contoh

def tambah(a, b):
  return a + b

hasil = tambah(2, 3)
print(hasil)

Kesimpulan

Return digunakan untuk mengambil hasil dari function.

Pengertian

Dictionary adalah tipe data yang menyimpan data dalam bentuk key dan value.

Penjelasan

Setiap data memiliki pasangan kunci (key) dan nilai (value).

Contoh

data = {
  "nama": "Angga",
  "umur": 15
}

print(data["nama"])

Kesimpulan

Dictionary digunakan untuk menyimpan data dengan pasangan key-value.

Pengertian

Tuple adalah tipe data seperti list tetapi tidak bisa diubah.

Penjelasan

Tuple bersifat immutable (tidak dapat diubah setelah dibuat).

Contoh

data = ("apel", "jeruk", "mangga")
print(data[0])

Kesimpulan

Tuple digunakan untuk data yang tidak perlu diubah.

Pengertian

Set adalah tipe data yang tidak memiliki urutan dan tidak bisa duplikat.

Penjelasan

Set digunakan untuk menyimpan data unik.

Contoh

data = {1, 2, 3, 3, 4}
print(data)

Kesimpulan

Set otomatis menghapus data yang duplikat.

Pengertian

Operator adalah simbol untuk melakukan operasi pada data.

Penjelasan

Contohnya seperti +, -, *, / untuk matematika.

Contoh

a = 5
b = 2
print(a + b)
print(a * b)

Kesimpulan

Operator digunakan untuk melakukan perhitungan dan operasi.

Pengertian

Operator perbandingan digunakan untuk membandingkan nilai.

Penjelasan

Hasilnya berupa True atau False.

Contoh

a = 5
b = 3
print(a > b)
print(a == b)

Kesimpulan

Operator ini digunakan dalam kondisi seperti if.

Pengertian

Operator logika digunakan untuk menggabungkan kondisi.

Penjelasan

Contohnya adalah and, or, dan not.

Contoh

umur = 18
print(umur > 17 and umur < 25)

Kesimpulan

Operator logika membantu membuat kondisi lebih kompleks.

Pengertian

String adalah tipe data untuk teks.

Penjelasan

String bisa menggunakan tanda kutip satu atau dua.

Contoh

teks = "Halo Bintang"
print(teks.upper())

Kesimpulan

String digunakan untuk menyimpan dan mengolah teks.

Pengertian

Method string adalah fungsi bawaan untuk mengolah string.

Penjelasan

Contohnya upper(), lower(), dan replace().

Contoh

teks = "halo"
print(teks.upper())
print(teks.replace("halo", "hai"))

Kesimpulan

Method memudahkan manipulasi teks.

Pengertian

File handling digunakan untuk membaca dan menulis file.

Penjelasan

Python bisa membuka file dengan mode read (r) dan write (w).

Contoh

file = open("data.txt", "w")
file.write("Halo Dunia")
file.close()

Kesimpulan

File handling memungkinkan program menyimpan data.

Pengertian

Try except digunakan untuk menangani error.

Penjelasan

Jika terjadi error, program tidak langsung berhenti.

Contoh

try:
  print(10 / 0)
except:
  print("Terjadi error")

Kesimpulan

Try except membuat program lebih aman dari error.

Pengertian

Class adalah blueprint untuk membuat object.

Penjelasan

Digunakan dalam konsep OOP (Object Oriented Programming).

Contoh

class Orang:
  nama = "Nita"

obj = Orang()
print(obj.nama)

Kesimpulan

Class digunakan untuk membuat struktur data berbasis objek.

Pengertian

Constructor adalah method yang dijalankan saat object dibuat.

Penjelasan

Biasanya digunakan untuk mengisi nilai awal.

Contoh

class Orang:
  def __init__(self, nama):
    self.nama = nama

obj = Orang("Rudi")
print(obj.nama)

Kesimpulan

Constructor mempermudah inisialisasi object.

Pengertian

Method adalah fungsi yang ada di dalam class.

Penjelasan

Method digunakan untuk menjalankan aksi pada object.

Contoh

class Orang:
  def sapa(self):
    print("Halo")

obj = Orang()
obj.sapa()

Kesimpulan

Method membuat object bisa melakukan aksi tertentu.

Pengertian

Inheritance adalah konsep OOP dimana sebuah class bisa mewarisi sifat dari class lain.

Penjelasan

Dengan inheritance, kita bisa menggunakan kembali kode dari class induk (parent class) ke class turunan (child class). Ini membuat program lebih efisien dan mengurangi pengulangan kode.

Contoh

class Orang:
  def sapa(self):
    print("Halo")

class Siswa(Orang):
  pass

obj = Siswa()
obj.sapa()

Kesimpulan

Inheritance membantu penggunaan ulang kode dan membuat struktur program lebih rapi.

Pengertian

Overriding adalah proses mengganti method dari parent class di child class.

Penjelasan

Jika child class memiliki method dengan nama yang sama seperti parent class, maka method tersebut akan menimpa (override) method yang ada di parent.

Contoh

class Orang:
  def sapa(self):
    print("Halo Orang")

class Siswa(Orang):
  def sapa(self):
    print("Halo Siswa")

obj = Siswa()
obj.sapa()

Kesimpulan

Overriding memungkinkan kita menyesuaikan perilaku class turunan.

Pengertian

Encapsulation adalah konsep menyembunyikan data dalam class agar tidak bisa diakses langsung.

Penjelasan

Biasanya menggunakan tanda underscore (_) atau double underscore (__) untuk menandai bahwa variabel bersifat private. Hal ini bertujuan untuk menjaga keamanan data.

Contoh

class Akun:
  def __init__(self, saldo):
    self.__saldo = saldo

  def lihat_saldo(self):
    print(self.__saldo)

obj = Akun(1000)
obj.lihat_saldo()

Kesimpulan

Encapsulation digunakan untuk melindungi data agar tidak sembarangan diakses.

Pengertian

Module adalah file Python yang berisi kode yang bisa digunakan di file lain.

Penjelasan

Dengan module, kita bisa memecah program besar menjadi beberapa bagian kecil agar lebih terstruktur. Module bisa di-import menggunakan keyword import.

Contoh

# file: hitung.py
def tambah(a, b):
  return a + b

# file utama
import hitung
print(hitung.tambah(2,3))

Kesimpulan

Module membantu membuat program lebih rapi dan mudah dikelola.

Pengertian

Library adalah kumpulan module yang sudah jadi dan bisa langsung digunakan.

Penjelasan

Python memiliki banyak library bawaan seperti math, datetime, dan random. Selain itu, ada juga library tambahan yang bisa diinstall seperti numpy atau pandas.

Contoh

import math

print(math.sqrt(16))

Kesimpulan

Library mempermudah pengembangan program tanpa harus membuat semuanya dari nol.

Pengertian

JSON adalah format data yang sering digunakan untuk pertukaran data antar aplikasi.

Penjelasan

JSON mirip dictionary di Python, mudah dibaca manusia dan bisa dikirim lewat internet. Cocok untuk menyimpan konfigurasi atau komunikasi API.

Contoh

import json

data = '{"nama": "Nadia", "umur": 16}'
obj = json.loads(data)
print(obj["nama"])

Kesimpulan

JSON memudahkan pertukaran data antar sistem dengan format yang ringan dan fleksibel.

Pengertian

Lambda adalah fungsi kecil tanpa nama (anonymous function).

Penjelasan

Digunakan untuk operasi sederhana dan cepat, biasanya dalam satu baris kode, contohnya untuk manipulasi list atau sebagai parameter fungsi lain.

Contoh

tambah = lambda a, b: a + b
print(tambah(5,3))

Kesimpulan

Lambda membuat kode lebih singkat dan praktis untuk operasi sederhana.

Pengertian

List comprehension adalah cara cepat membuat list baru dari list lama.

Penjelasan

Dengan sintaks ringkas, kita bisa melakukan filtering atau manipulasi elemen sekaligus saat membuat list baru.

Contoh

angka = [1,2,3,4,5]
kuadrat = [x**2 for x in angka]
print(kuadrat)

Kesimpulan

List comprehension membuat kode lebih ringkas dan efisien dibandingkan loop biasa.

Pengertian

Decorator adalah fungsi yang menambahkan fungsionalitas ke fungsi lain tanpa mengubah kode asli.

Penjelasan

Decorator biasanya dipakai untuk logging, validasi, atau akses kontrol. Ditulis dengan simbol @ di atas fungsi.

Contoh

def dekor(fungsi):
  def wrapper():
    print("Sebelum fungsi")
    fungsi()
    print("Sesudah fungsi")
  return wrapper

@dekor
def halo():
  print("Halo Dunia")

halo()

Kesimpulan

Decorator membuat kode lebih modular dan mudah menambahkan fitur baru ke fungsi.

Pengertian

Generator adalah fungsi yang menghasilkan urutan nilai secara satu per satu.

Penjelasan

Dengan generator, kita bisa menghemat memori karena tidak membuat seluruh list sekaligus. Generator sangat cocok untuk data besar atau streaming data.

Contoh

def hitung():
  for i in range(5):
    yield i

for x in hitung():
  print(x)

Kesimpulan

Generator mempermudah pengelolaan data besar tanpa membebani memori.

Pengertian

Kita bisa membuat exception sendiri untuk menangani kondisi khusus.

Penjelasan

Custom exception berguna agar program lebih jelas saat error tertentu terjadi, misal input invalid atau kondisi bisnis tertentu.

Contoh

class ErrorUmur(Exception):
    pass

umur = -5
if umur < 0:
    raise ErrorUmur("Umur tidak boleh negatif")

Kesimpulan

Custom exception membantu membuat error handling lebih spesifik dan aman.

Pengertian

Regex adalah pola untuk mencari atau memanipulasi teks.

Penjelasan

Regex berguna untuk validasi input, pencarian teks, atau ekstraksi data secara efisien.

Contoh

import re

text = "Email saya: anton@mail.com"
pattern = r'\S+@\S+'
hasil = re.findall(pattern, text)
print(hasil)

Kesimpulan

Regex memudahkan pencarian pola teks kompleks dalam string.

Pengertian

Fungsi map, filter, dan reduce digunakan untuk memproses data secara fungsional.

Penjelasan

Map untuk transformasi data, filter untuk seleksi, reduce untuk menggabungkan data menjadi satu nilai.

Contoh

from functools import reduce

angka = [1,2,3,4,5]
kuadrat = list(map(lambda x: x**2, angka))
genap = list(filter(lambda x: x%2==0, angka))
total = reduce(lambda a,b: a+b, angka)

print(kuadrat, genap, total)

Kesimpulan

Map, filter, reduce mempermudah manipulasi list secara fungsional.

Pengertian

Virtual environment adalah lingkungan Python terisolasi.

Penjelasan

Dengan virtual environment, kita bisa menginstal library berbeda untuk proyek berbeda tanpa konflik versi.

Contoh

# buat environment
python -m venv env

# aktifkan
env\Scripts\activate  # Windows
source env/bin/activate  # Linux/Mac

Kesimpulan

Virtual environment menjaga proyek Python tetap terpisah dan rapi.

Pengertian

pip adalah tool untuk mengelola library Python.

Penjelasan

Dengan pip kita bisa install, upgrade, dan uninstall library pihak ketiga untuk proyek kita.

Contoh

# install library
pip install requests

# upgrade library
pip install --upgrade requests

# uninstall library
pip uninstall requests

Kesimpulan

pip mempermudah manajemen library untuk pengembangan proyek Python.

Pengertian

Modul adalah file Python yang berisi kode seperti fungsi, class, atau variabel.

Penjelasan

Kita bisa memisahkan kode menjadi modul agar lebih rapi. Dengan import, modul bisa digunakan di file lain tanpa menulis ulang kode.

Contoh

# file math_utils.py
def tambah(a, b):
    return a + b

# file main.py
import math_utils
print(math_utils.tambah(5, 7))

Kesimpulan

Modul mempermudah pengelolaan kode dan penggunaan ulang fungsi di banyak file.

Pengertian

OS module adalah modul bawaan Python untuk berinteraksi dengan sistem operasi.

Penjelasan

Dengan OS module kita bisa membaca file, folder, path, dan informasi sistem. Cocok untuk scripting otomatis.

Contoh

import os

print(os.getcwd())  # Cek folder kerja sekarang
print(os.listdir())  # List file di folder

Kesimpulan

OS module memudahkan manipulasi sistem file dan direktori dari Python.

Pengertian

Sys module adalah modul untuk berinteraksi dengan Python runtime environment.

Penjelasan

Sys module berguna untuk mengakses argumen command line, versi Python, atau keluar dari program.

Contoh

import sys

print(sys.version)
print(sys.argv)

Kesimpulan

Sys module membantu kontrol program dan integrasi dengan command line.

Pengertian

Time module digunakan untuk mengatur waktu, delay, dan timestamp.

Penjelasan

Time module sering dipakai untuk menghentikan program sementara atau mengukur lama eksekusi.

Contoh

import time

print("Mulai")
time.sleep(2)  # Delay 2 detik
print("Selesai")

Kesimpulan

Time module memudahkan manipulasi waktu dan delay program.

Pengertian

Random module digunakan untuk menghasilkan nilai acak.

Penjelasan

Biasanya dipakai untuk simulasi, game, atau pengambilan sampel data acak.

Contoh

import random

print(random.randint(1, 10))  # Angka acak 1-10
print(random.choice(["Rudi","Lina","Anton"]))

Kesimpulan

Random module membantu membuat program lebih dinamis dan tidak deterministik.

Pengertian

JSON module digunakan untuk parsing JSON dan konversi data Python ke JSON.

Penjelasan

Cocok untuk menyimpan konfigurasi, pertukaran data dengan web API, atau menyimpan data ke file.

Contoh

import json

data = {"nama":"Sari","umur":17}
json_str = json.dumps(data)
print(json_str)

Kesimpulan

JSON module mempermudah konversi antara Python dan format JSON.

Pengertian

Datetime module digunakan untuk manipulasi tanggal dan waktu.

Penjelasan

Kita bisa mendapatkan tanggal sekarang, membuat tanggal tertentu, dan menghitung selisih tanggal.

Contoh

import datetime

now = datetime.datetime.now()
print(now)
lahir = datetime.datetime(2008, 5, 12)
print(now - lahir)

Kesimpulan

Datetime module memudahkan kerja dengan tanggal dan waktu di Python.

Pengertian

Collections module menyediakan tipe data khusus seperti deque, Counter, dan defaultdict.

Penjelasan

Cocok untuk kasus khusus seperti hitung frekuensi, antrian, atau default value dictionary.

Contoh

from collections import Counter

data = ["apel","jeruk","apel","mangga"]
hitung = Counter(data)
print(hitung)

Kesimpulan

Collections module mempermudah pengolahan data kompleks dengan struktur siap pakai.

Pengertian

Itertools module menyediakan fungsi untuk iterasi canggih dan kombinatorial.

Penjelasan

Cocok untuk membuat kombinasi, permutasi, dan iterasi tak terbatas tanpa banyak memori.

Contoh

import itertools

angka = [1,2,3]
kombinasi = list(itertools.combinations(angka, 2))
print(kombinasi)

Kesimpulan

Itertools mempermudah manipulasi data kompleks secara efisien.

Pengertian

Logging module digunakan untuk mencatat jalannya program.

Penjelasan

Dengan logging, kita bisa mencatat informasi, warning, atau error tanpa menghentikan program. Cocok untuk debugging dan monitoring aplikasi.

Contoh

import logging

logging.basicConfig(level=logging.INFO)
logging.info("Program dimulai")
logging.warning("Ini peringatan")

Kesimpulan

Logging module membantu memantau jalannya program dan mendiagnosa masalah.
