Pilih Bagian

Kembali Pilih Materi

Pengertian

HTML adalah bahasa dasar untuk membuat website.

Penjelasan

HTML digunakan untuk menyusun struktur seperti teks, gambar, dan link.

Contoh

<h1>Halo Dunia</h1>

Kesimpulan

HTML adalah pondasi dari semua website.

Pengertian

Struktur HTML adalah susunan dasar halaman web.

Penjelasan

Terdiri dari html, head, dan body.

Contoh

<html>
<body>Halo</body>
</html>

Kesimpulan

Struktur HTML wajib ada di setiap halaman.

Pengertian

Tag digunakan untuk membuat elemen HTML.

Penjelasan

Contohnya h1 untuk judul dan p untuk paragraf.

Contoh

<h1>Judul</h1>
<p>Paragraf</p>

Kesimpulan

Tag adalah bagian utama dalam HTML.

Pengertian

HTML bisa menampilkan link dan gambar.

Penjelasan

Gunakan tag a untuk link dan img untuk gambar.

Contoh

<a href="#">Klik</a>
<img src="gambar.jpg">

Kesimpulan

HTML bisa menghubungkan halaman dan menampilkan media.

Pengertian

Form digunakan untuk input data dari user.

Penjelasan

Biasanya digunakan untuk login atau pendaftaran.

Contoh

<input type="text">

Kesimpulan

Form digunakan untuk interaksi dengan user.

Pengertian

List digunakan untuk membuat daftar item.

Penjelasan

Ada ordered list (angka) dan unordered list (bullet).

Contoh

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Kesimpulan

List mempermudah menampilkan data terstruktur.

Pengertian

Tabel digunakan untuk menampilkan data baris & kolom.

Penjelasan

Gunakan tag table, tr, dan td.

Contoh

<table>
  <tr><td>A</td><td>B</td></tr>
</table>

Kesimpulan

Tabel berguna untuk data yang terorganisir.

Pengertian

HTML bisa menampilkan audio dan video.

Penjelasan

Gunakan tag audio untuk suara dan video untuk video.

Contoh

<audio src="musik.mp3" controls></audio>
<video src="video.mp4" controls></video>

Kesimpulan

HTML mendukung multimedia interaktif.

Pengertian

Semantic HTML membuat kode lebih mudah dimengerti.

Penjelasan

Gunakan header, nav, article, footer, dll.

Contoh

<header>Judul</header>
<footer>Hak Cipta</footer>

Kesimpulan

Semantic HTML baik untuk SEO & keterbacaan.

Pengertian

Meta tag memberi info tambahan tentang halaman.

Penjelasan

Contohnya deskripsi, keyword, viewport untuk SEO & responsif.

Contoh

<meta name="description" content="Website belajar">

Kesimpulan

Meta penting untuk optimasi mesin pencari & tampilan di device.

Pengertian

Button digunakan untuk aksi klik user.

Penjelasan

Bisa dipakai submit form atau interaksi JavaScript.

Contoh

<button>Klik</button>

Kesimpulan

Button memudahkan user memberi perintah.

Pengertian

Iframe menampilkan halaman web lain di halaman kita.

Penjelasan

Bisa dipakai untuk embed video atau website.

Contoh

<iframe src="https://example.com"></iframe>

Kesimpulan

Iframe memudahkan menampilkan konten eksternal.

Pengertian

Label & fieldset mempermudah form lebih terstruktur.

Penjelasan

Label untuk keterangan input, fieldset untuk grup input.

Contoh

<fieldset>
  <label>Nama</label>
  <input type="text">
</fieldset>

Kesimpulan

Membuat form lebih jelas & user-friendly.

Pengertian

Div & span untuk pengelompokan konten.

Penjelasan

Div untuk blok, span untuk inline.

Contoh

<div>Blok</div>
<span>Teks</span>

Kesimpulan

Membantu styling & layout halaman.

Pengertian

Comment untuk catatan di kode HTML.

Penjelasan

Tidak muncul di browser, hanya untuk developer.

Contoh

<!-- Ini komentar -->

Kesimpulan

Comment membantu dokumentasi & penjelasan kode.

Pengertian

Style inline menambahkan CSS langsung di tag HTML.

Penjelasan

Cocok untuk styling cepat tapi kurang efisien.

Contoh

<p style="color:red">Teks merah</p>

Kesimpulan

Inline style cepat tapi tidak direkomendasikan untuk proyek besar.

Pengertian

CSS eksternal memisahkan styling dari HTML.

Penjelasan

Lebih rapi & bisa dipakai di banyak halaman.

Contoh

<link rel="stylesheet" href="style.css">

Kesimpulan

CSS eksternal lebih efisien untuk proyek besar.

Pengertian

Class & ID untuk menandai elemen HTML.

Penjelasan

Class bisa dipakai banyak elemen, ID unik untuk satu elemen.

Contoh

<div class="box"></div>
<p id="judul">Halo</p>

Kesimpulan

Memudahkan styling & scripting elemen HTML.

Pengertian

Tabel lanjutan untuk header & footer.

Penjelasan

Gunakan thead, tbody, tfoot untuk struktur lebih baik.

Contoh

<table>
  <thead><tr><th>Nama</th></tr></thead>
  <tbody><tr><td>Budi</td></tr></tbody>
</table>

Kesimpulan

Membuat tabel lebih terstruktur & mudah dibaca.

Pengertian

HTML entities untuk menampilkan karakter khusus.

Penjelasan

Contohnya & untuk & atau < untuk <.

Contoh

<p>5 & 6</p>

Kesimpulan

Membantu menampilkan karakter khusus tanpa error.

Pengertian

Audio bisa dikontrol lebih lanjut dengan atribut tambahan.

Penjelasan

Gunakan autoplay, loop, muted, dan controls untuk pengalaman audio lebih baik.

Contoh

<audio src="musik.mp3" controls autoplay loop muted></audio>

Kesimpulan

Membuat audio lebih interaktif dan nyaman untuk user.

Pengertian

Video bisa dikontrol lebih lengkap seperti audio.

Penjelasan

Atribut autoplay, loop, muted, poster memberi pengalaman video lebih baik.

Contoh

<video src="video.mp4" controls autoplay loop muted poster="cover.jpg"></video>

Kesimpulan

Video lebih interaktif & menarik dengan kontrol tambahan.

Pengertian

Select & option membuat input berupa dropdown.

Penjelasan

Membantu user memilih pilihan yang sudah disediakan.

Contoh

<select>
  <option>Pilihan 1</option>
  <option>Pilihan 2</option>
</select>

Kesimpulan

Dropdown mempermudah input pilihan user secara cepat.

Pengertian

Checkbox & radio untuk memilih opsi tunggal atau banyak.

Penjelasan

Checkbox bisa pilih banyak, radio hanya satu per grup.

Contoh

<input type="checkbox">Option A
<input type="radio" name="group1">Option B

Kesimpulan

Mempermudah user memilih input sesuai kebutuhan.

Pengertian

Textarea untuk input teks panjang seperti komentar.

Penjelasan

Bisa diubah ukurannya dan memberi ruang lebih untuk mengetik.

Contoh

<textarea rows="4" cols="50">Tulis komentar</textarea>

Kesimpulan

Textarea mempermudah input teks panjang dari user.

Pengertian

Header semantik untuk judul & navigasi utama halaman.

Penjelasan

Memudahkan pembaca & mesin pencari memahami struktur halaman.

Contoh

<header>
  <h1>Judul Website</h1>
  <nav>Menu</nav>
</header>

Kesimpulan

Meningkatkan SEO dan keterbacaan halaman.

Pengertian

Article & section membantu membagi konten menjadi bagian logis.

Penjelasan

Article untuk konten utama, section untuk sub-bagian dengan topik sama.

Contoh

<article>
  <section>Isi section</section>
</article>

Kesimpulan

Membuat konten terstruktur & mudah dipahami pembaca.

Pengertian

Footer semantik untuk informasi penutup halaman.

Penjelasan

Biasanya berisi hak cipta, kontak, atau link penting.

Contoh

<footer>Hak Cipta 2026</footer>

Kesimpulan

Footer membantu navigasi & memberikan informasi penting di bagian bawah halaman.

Pengertian

Meta viewport membuat halaman responsif di berbagai device.

Penjelasan

Biasanya digunakan untuk menyesuaikan lebar halaman di mobile.

Contoh

<meta name="viewport" content="width=device-width, initial-scale=1">

Kesimpulan

Membuat website tampil baik di PC maupun smartphone.

Pengertian

Favicon adalah ikon kecil di tab browser.

Penjelasan

Memberikan identitas visual website & memudahkan pengunjung mengenali halaman.

Contoh

<link rel="icon" href="favicon.ico">

Kesimpulan

Favicon membuat website lebih profesional & mudah dikenali.

Pengertian

Button bisa dikombinasi dengan atribut untuk interaksi lebih kompleks.

Penjelasan

Gunakan type="submit", "reset", "button" dan event JavaScript untuk aksi.

Contoh

<button type="submit">Kirim</button>

Kesimpulan

Button lebih fleksibel dengan atribut dan scripting.

Pengertian

Input HTML bisa beragam tipe untuk kebutuhan spesifik.

Penjelasan

Contohnya email, password, number, date, range, checkbox, radio.

Contoh

<input type="email">
<input type="number">

Kesimpulan

Input tipe spesifik membantu validasi & pengalaman user lebih baik.

Pengertian

Validation memeriksa input sebelum dikirim ke server.

Penjelasan

Gunakan required, pattern, min, max untuk validasi sederhana.

Contoh

<input type="text" required pattern="[A-Za-z]+">

Kesimpulan

Validation mencegah input salah & meningkatkan keamanan.

Pengertian

Input file memungkinkan user mengunggah file ke website.

Penjelasan

Gunakan input type="file" & accept untuk tipe file tertentu.

Contoh

<input type="file" accept=".jpg,.png">

Kesimpulan

Membuat user bisa mengirim file seperti foto atau dokumen.

Pengertian

Audio & video bisa di-embed dari sumber eksternal.

Penjelasan

Gunakan iframe atau tag embed dari YouTube, Spotify, dll.

Contoh

<iframe src="https://www.youtube.com/embed/videoid"></iframe>

Kesimpulan

Mempermudah menambahkan media tanpa upload langsung ke server.

Pengertian

Meta tags memberi info tambahan seperti charset, viewport, author.

Penjelasan

Membantu SEO, tampilan mobile, dan kompatibilitas browser.

Contoh

<meta charset="UTF-8">
<meta name="author" content="Bintang">

Kesimpulan

Meta tags penting untuk performa dan keterbacaan halaman.
